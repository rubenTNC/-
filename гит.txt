команты гита 

git init - создать репозиторий

git add - добавление файлов в отслеживание (название файла)

git status - позваоляет отследить текущий статус нашего репозитория 

git add * - добавляет все файлы в отслеживание 

git commit -m "название комиита" - создает коммит

git commit -a -m сразу добавляет измениня в репозиторий
